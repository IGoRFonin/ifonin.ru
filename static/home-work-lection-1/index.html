Поставим Claude Code с нуля за один вечер

15 шагов, отдельные ветки для Windows и macOS. Прогресс сохраняется в браузере — можно прерваться и вернуться туда, где остановился.

⏱ 30–60 мин ◆ 15 шагов 💬 Если что — пиши в телеграм
I
Часть первая

Аккаунт и подписка

00
Шаг 00

Поставь VPN

Сайт claude.ai в России заблокирован. Без VPN не получится ни зарегистрироваться, ни скачать Claude Desktop, ни пройти первую авторизацию в Claude Code.

Подойдёт любой рабочий VPN — какой уже стоит, тот и используй. Если нет вообще ничего или хочется вместо VPN прокси — напиши мне в телеграм, подскажу.

Важно
VPN нужен всегда, когда работаешь с Claude. Включай, прежде чем запускать claude в терминале.
01
Шаг 01

Зарегистрируйся на claude.ai

Включай VPN. Открывай claude.ai, жми Sign up, вбивай свой email. На почту прилетит код — вводи его.

На следующем экране попросят номер телефона. Вкладку не закрывай — номер сейчас купим.

Если что-то не получается — пиши мне в телеграм.

02
Шаг 02

Возьми виртуальный номер на Grizzly SMS

В новой вкладке открывай grizzlysms.com/ru/claude. Регистрируйся, закинь на баланс рублей 30–50 — обычно этого хватает.

Выбирай страну (я обычно беру Индонезию — дешевле и стабильно работает), жми «Получить». Номер появится на вкладке «Полученные номера».

Если не получается — пиши мне в телеграм, разберёмся.
03
Шаг 03

Подтверди аккаунт через номер

Копируй номер из Grizzly SMS, вставляй его в форму на claude.ai, жми «Отправить SMS».

Возвращайся на вкладку Grizzly SMS — в течение минуты в столбце «Код» появится 6 цифр. Вбивай их на claude.ai.

Если код не пришёл минут за 20 — деньги сами вернутся на баланс, купи другой номер. Застрял — пиши в телеграм.
04
Шаг 04

Купи подписку Claude Pro

Открывай plati.market. Бери PRO (1 месяц) — для старта хватит. Платишь картой РФ.

После оплаты продавец пришлёт инструкцию — обычно это ссылка, по которой подписка привяжется к твоему email от claude.ai.

05
Шаг 05

Проверь, что подписка активна

Заходи на claude.ai под VPN. В левом нижнем углу должно стоять Pro (а не Free).

Отправь любое тестовое сообщение: «привет, ты работаешь?». Ответ пришёл — отлично, идём дальше.

06
Шаг 06

Скачай Claude Desktop

Под VPN открывай claude.ai/download. Сайт сам поймёт, какая у тебя система, и даст нужный установщик.

Ставь, запускай, логинься под тем же аккаунтом. Проверь, что чат отвечает — значит всё ок.

II
Часть вторая

Терминал и Node.js

Шаги 07–10 зависят от системы
07
Win · 07

Скачай Git for Windows

Открывай git-scm.com/download/win. Скачивание стартанёт само (файл Git-…-64-bit.exe).

Запускай установщик, на всех шагах жми Next. На экране «Choosing the default editor» выбери Notepad вместо Vim — так проще, если только начинаешь.

08
Win · 08

Запусти Git Bash и проверь, что работает

Меню Пуск → Git Bash. Откроется чёрное окно с приглашением вида username@DESKTOP MINGW64 ~ $.

Вбивай команду и жми Enter:

pwd

Должен вывестись путь к домашней папке (типа /c/Users/Имя) — значит терминал живой.

09
Win · 09

Поставь Node.js

Node.js нужен, чтобы работал сам Claude Code — он написан на JavaScript и запускается через Node. Без Node его просто нечем запускать. Ещё в комплекте идёт npm — через него мы и поставим Claude.

Открывай nodejs.org/en/download. Бери LTS и Windows Installer (.msi), x64. Ставь.

Важно: закрой Git Bash и открой заново — иначе терминал не увидит Node. Потом проверяй:

node --version
npm --version

Обе команды должны вывести номер версии. Если выводят — едем дальше.

10
Win · 10

Поставь Claude Code

В Git Bash вбивай:

npm install -g @anthropic-ai/claude-code

Когда установка добежит до конца, проверь:

claude --version

Вывелась версия — поздравляю, Claude Code у тебя на компе.

07
Mac · 07

Скачай cmux

Открывай прямую ссылку на DMG: cmux-macos.dmg.

Двойной клик по DMG → перетаскивай cmux в Applications.

cmux — удобный терминал под AI-агентов. Можно обойтись и стандартным Terminal.app, но в cmux приятнее.
08
Mac · 08

Запусти cmux и проверь, что работает

Открывай через Launchpad или Spotlight (⌘+Space → cmux). При первом запуске macOS спросит — жми Open.

pwd

Путь типа /Users/Имя — значит работает.

09
Mac · 09

Поставь Node.js

Node.js нужен, чтобы работал сам Claude Code — он написан на JavaScript и запускается через Node. Без Node его нечем запустить. Заодно появится npm — через него мы и установим Claude.

Открывай nodejs.org/en/download. Бери LTS и macOS Installer (.pkg).

Запускай, везде Continue и Install, введи пароль от Mac. Закрой cmux и открой его заново — иначе не увидит Node. Потом проверяй:

node --version
npm --version

Обе команды должны вывести номер версии. Вывели — хорошо.

10
Mac · 10

Поставь Claude Code

В cmux вбивай:

npm install -g @anthropic-ai/claude-code

Когда установка дойдёт до конца, проверь:

claude --version

Вывелась версия — готово, Claude Code у тебя на маке.

III
Часть третья

Первый сайт через Claude Code

11
Шаг 11

Создай папку под первый проект

В терминале вбивай:

mkdir my-first-landing
cd my-first-landing

Первая команда создала папку, вторая — зашла в неё. Все файлы, которые напишет Claude, лягут сюда.

12
Шаг 12

Запусти Claude Code и авторизуйся

Включай VPN. Он нужен для авторизации и будет нужен дальше.

claude

Откроется приглашение Claude. В нём вбивай:

/login

Откроется браузер. Входи в свой аккаунт, разрешай доступ. Как увидел «Successfully authenticated» — возвращайся в терминал.

13
Шаг 13

Альтернатива VPN — прокси (опционально)

Если хочешь вместо VPN использовать прокси — напиши мне в телеграм, расскажу, как настроить.

14
Шаг 14

Напиши первый промпт

В запущенном claude вставляй и жми Enter:

создай лендинг по предоставлению услуг по разработке сайтов. Сделай в лучших подходах по маркетингу. В светлых тонах

Claude либо задаст пару уточняющих вопросов, либо сразу пойдёт писать файлы. Когда спросит разрешение на создание файлов — соглашайся.

15
Шаг 15

Открой готовую страницу в браузере

Когда Claude закончит работать — в папке my-first-landing появится HTML-файл (обычно index.html).

Открывай Finder (на маке) или Проводник (на винде), иди в папку my-first-landing, двойной клик по index.html — страница откроется в браузере. Поздравляю, твой первый сайт готов.

● Завершено

Claude Code работает. Первый сайт готов.

Если на каком-то шаге застрял или что-то не взлетело — пиши мне в телеграм, разберёмся вместе.
