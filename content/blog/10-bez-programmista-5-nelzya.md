Меня регулярно спрашивают знакомые - предприниматели, маркетологи, продакты - "а вот это я могу сделать сам, без разработчика?" Я программист, и раньше ответ был почти всегда "нет, давай я помогу". Сейчас всё чаще отвечаю "да, справишься".

Один инструмент - Claude Code. Это терминальный AI-ассистент, которому описываешь задачу на обычном русском языке, а он пишет код, создаёт файлы, деплоит. И скиллы к нему - наборы инструкций, которые учат его делать конкретные вещи хорошо. Не просто "напиши HTML", а "сделай лендинг с правильной типографикой, отступами и адаптивной верстой под мобильные". Без скиллов получаешь код. Со скиллами - результат, который не стыдно показать.

Порог входа минимальный: умеешь объяснить что хочешь словами - достаточно.

---

## Что реально работает

### 1. Лендинг с нормальным дизайном

Говоришь "сделай лендинг для продукта X, вот описание" - получаешь готовый сайт. Скиллы для дизайна делают результат не хуже студии за несколько десятков тысяч рублей: правильные отступы, читаемая типографика, адаптив под телефоны.

Выходной файл - чистый HTML или React. Можно захостить где угодно - на Vercel бесплатно, за несколько минут.

Я так сделал лендинг для ifonin.ru - и это задача, с которой справится человек без опыта в коде.

### 2. Telegram-бот

"Сделай бота, который принимает заявки через форму и пересылает мне в личку" - рабочий код через час. Claude Code напишет логику, объяснит как зарегистрировать бота через BotFather, поможет разобраться с деплоем.

Если застрял - объясняешь проблему в чат, он разбирается и чинит. Я видел как знакомые без технического бэкграунда запускали ботов за вечер - и удивлялись сами себе.

### 3. Полноценный сайт с несколькими страницами

Не одностраничник, а сайт с навигацией, блогом, разными секциями. Claude Code сам выберет подходящий инструмент - Hugo, Astro, Next.js - в зависимости от задачи. Скиллы обеспечат современный внешний вид без того, чтобы самому разбираться в CSS.

Этот блог - тоже результат работы с Claude Code.

### 4. MVP и рабочий прототип

Трекер задач, калькулятор стоимости, генератор чего угодно, простой дашборд. Claude Code собирает React-компоненты, подключает базу данных, делает API. Получается рабочий прототип - не макет в Figma, а кликабельный сервис, который можно показать первым пользователям.

Раньше для этого нанимали разработчика. Теперь можно сначала проверить идею, а разработчика звать когда понятно, что продукт нужен.

### 5. Автоматизация рутины

"Каждое утро собирай данные из Google Sheets и отправляй сводку мне в Telegram" - Claude Code напишет Python-скрипт, настроит расписание. Один раз описал задачу, один раз запустил - дальше работает само.

Для простых цепочек это полноценная замена платным сервисам автоматизации.

### 6. Парсер для сбора данных

Мониторинг цен конкурентов, сбор объявлений, отслеживание упоминаний. Claude Code напишет парсер под конкретный сайт, настроит хранение результатов, добавит уведомления когда что-то изменилось.

Это задачи, за которые фрилансеры берут от нескольких тысяч рублей каждый раз.

### 7. Интеграции между сервисами

Подключить форму на сайте к Google Sheets и Telegram одновременно. Настроить вебхук между двумя API. Связать разные сервисы между собой через промежуточный код.

Claude Code умеет работать с API, писать middleware, разбираться в документации незнакомых сервисов - и объяснять что происходит человеческим языком.

### 8. Форма с логикой

Анкета, где следующий вопрос зависит от предыдущего ответа. Сохранение результатов в таблицу. Отправка уведомления при заполнении. Валидация полей.

Скиллы для дизайна делают такие формы визуально приличными - не голый HTML из девяностых.

### 9. HTML-шаблоны для писем

Транзакционные письма, шаблоны рассылки, онбординговые цепочки. Claude Code напишет и HTML-шаблон письма, который нормально отображается в Gmail и Outlook, и код для отправки через SMTP или API любого почтового сервиса.

### 10. Чат-бот поддержки по своей базе знаний

Бот, который отвечает на вопросы клиентов на основе ваших материалов - FAQ, документации, описания продукта. Claude Code подключит нужный AI API, настроит промпты, сделает интерфейс. Встраивается в сайт или работает в Telegram.

## Где заканчивается зона Claude Code

### 1. Высоконагруженные системы

Claude Code напишет код, но не спроектирует архитектуру для тысяч одновременных пользователей. Кеширование, очереди задач, шардирование базы данных, настройка реплик - это не про генерацию кода, это про опыт построения систем. Разница между кодом, который работает у пяти пользователей, и кодом, который работает у пятидесяти тысяч - архитектурная, не синтаксическая.

### 2. Сложная бизнес-логика с граничными случаями

Подписки с пробными периодами и возвратами. Расчёт стоимости доставки с десятком условий. Мультивалютность с пересчётом в реальном времени. AI хорошо пишет "счастливый путь" - когда всё идёт по плану. Граничные случаи, исключения, конкурентные запросы - здесь Claude Code ведёт себя как блестящий инженер с амнезией: починит проблему в одном месте, создаст её в другом. Это не метафора - именно так описывают этот паттерн разработчики, которые работают с AI на больших проектах.

### 3. Интеграции с 1С и legacy-системами

Кастомные протоколы обмена. Нестандартные форматы файлов. API с документацией на бумаге двенадцатилетней давности. Тут нужен человек с конкретным опытом конкретной системы - никакое обучение AI это не заменит.

### 4. Всё, где цена ошибки высокая

Платёжные системы. Персональные данные в рамках 152-ФЗ. Медицинские записи. Финансовые транзакции. Здесь нужен аудит кода живым специалистом, а не его генерация. Известны случаи, когда AI-код для работы с криптографией содержал уязвимость, которая стоила реальных денег. Экономия на разработчике в таких местах обходится дороже самого разработчика.

### 5. Поддержка большого проекта

Когда кодовая база выросла и перестала помещаться в контекстное окно за один раз - нужен человек, который держит в голове архитектуру целиком. Claude Code отлично справляется с задачей в рамках одной сессии, но не накапливает знания о проекте автоматически. Без скиллов, которые передают контекст, и без документации, которую поддерживаешь сам, большой проект превращается в набор плохо связанных кусков кода.

## Как это использовать

Claude Code закрывает большинство задач на старте: лендинг, бот, прототип, автоматизации. Если нужно проверить идею - это самый быстрый путь от мысли до работающего продукта без бюджета на разработку.

Скиллы - ключевая часть. Без них Claude Code пишет код, который работает, но выглядит как черновик. Со скиллами для дизайна и верстки результат можно сразу показывать людям.

Граница примерно такая: пока проект небольшой и задачи понятные - справляешься сам. Когда появляется реальная нагрузка, сложная логика или чувствительные данные - закладывай бюджет на разработчика. Лучше знать это до старта, чем узнавать в процессе.
