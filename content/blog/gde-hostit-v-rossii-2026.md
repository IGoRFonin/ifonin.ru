Vercel и Heroku закрыты, Netlify недоступна, а Aeza исчезла. Если ты держишь пет-проект на заграничном хостинге и думаешь "авось пока работает" — время мигрировать.

## Почему блокировки случились и почему это коснулось именно тебя

2024 год начался с тихих блокировок. Vercel перестал принимать российские карты. Потом Heroku. Netlify.

В 2025 блокировки стали массовыми: DigitalOcean, GoDaddy, Linode. Роскомнадзор не шутил. Закон о "приземлении" вступил в силу.

Я думал, что VPN поможет. Не помог. Платежи блокируются даже через VPN. Карты отклоняются. Сохранённые способы оплаты перестают работать без предупреждения — утром просто письмо "Your payment failed".

Закон 152-ФЗ требует локализации персональных данных россиян на серверах в России. Для бизнеса это не рекомендация, а обязанность. Штрафы серьёзные.

Русские провайдеры — единственная опция для стабильной работы. Vercel Pro стоит примерно $20/мес + overage за bandwidth. Timeweb VDS — около 500 рублей. Разница в цене ощутимая.

## С чего начать: первый проект на российском хостинге

**App Platform (Timeweb, ONREZA)** — если у тебя Next.js, React, Hugo, статический сайт. 2 клика, и всё работает.

**VDS (Timeweb, FirstVDS, AdminVPS)** — если нужен Docker, контроль, кастомная настройка. Telegram-бот, backend на Node.js, Python, full-stack приложение. Требуется базовый Linux.

**Виртуальный хостинг (Beget, reg.ru)** — WordPress, простой HTML/CSS/JS сайт. Панель управления, без командной строки.

Для старта нужно 5 минут. Домен (если еще нет). SSH-ключ (если выбрал VDS). Карта для оплаты.

Миграция с Vercel обычно занимает 1-2 часа, включая DNS-настройку. Hugo или статический сайт — 30-40 минут через rsync. Next.js с Docker — час-полтора. Больше всего времени уходит на проверку работы.

Мой блог ifonin.ru живет на Timeweb. Пушу изменения в GitHub. Coolify подхватывает webhook, собирает Hugo, деплоит. Изменения live через минуту — проверял секундомером. Автоматический HTTPS через Let's Encrypt, DDoS-защита включена.

## Чего НЕ делать: опасные ошибки

**VPN как решение**

Нестабилен. Платежи блокируются независимо от VPN. Vercel перестал принимать карты, даже через VPN. Сохранённые способы оплаты отклоняются без предупреждения.

В 2 часа ночи приходит письмо "Your usage exceeded limits", а ты не можешь оплатить. Такое случалось с моими знакомыми на reg.ru, когда блокировка застала врасплох.

**Дешевый shared hosting**

TTFB 2000+ миллисекунд — Google Core Web Vitals такое не прощает. Переход с дешевого shared на VPS часто улучшает позиции в поиске. Скорость — один из факторов ранжирования.

**Игнорирование 152-ФЗ**

Если у тебя бизнес, данные клиентов должны быть в России. Это закон. Штрафы серьёзные, блокировка сайта. Selectel, Timeweb, Beget — все соответствуют 152-ФЗ. Проверь, что твой провайдер тоже.

**"Я справлюсь сам" без опыта Linux**

Если не знаешь Linux, начни с Timeweb App Platform или Beget, не с raw VPS. Управление сервером — это SSH, nginx, firewall, обновления безопасности. Если ошибёшься, сайт может упасть.

App Platform делает всё за тебя. Потом научишься, перейдёшь на VDS.

## Миграция: проще, чем кажется

Откладывать миграцию — плохая стратегия.

**Статический сайт (Hugo, Jekyll, Astro)**

Собираешь сайт локально (`hugo`, `jekyll build`, `npm run build`). Заливаешь на сервер через rsync или scp. Настраиваешь nginx. Let's Encrypt для HTTPS, обновляешь DNS-записи, ждёшь propagation. Проверяешь. 30-40 минут с учётом DNS.

**Next.js / React / Node.js**

Coolify делает почти всё за тебя — но тебе нужно настроить сервер, добавить environment variables, подключить базу данных (если есть), настроить домен. Подключаешь GitHub репозиторий, выбираешь Docker или Nixpacks (автоматический билд). Деплоишь.

Coolify сам настроит nginx, HTTPS, webhook для автодеплоя. Час-полтора.

**WordPress**

Для новичка — самый простой вариант. Beget, Timeweb, reg.ru дают автоматическую установку WordPress через панель. Один клик. Потом импортируешь базу данных, загружаешь файлы через FTP.

30 минут максимум.

**База данных**

Экспортируешь дамп. Заливаешь на новый сервер. Меняешь connection string в .env файле. Проверяешь, что всё работает.

Если используешь PostgreSQL или MySQL — миграция почти всегда без проблем.

Знакомый переехал с Vercel Pro на российский VPS в декабре 2025. День работы, включая тестирование и откат старого окружения на случай проблем. Экономия в 4 раза, полный контроль, CI/CD через GitHub Actions.

## Особые случаи: Selectel и VDSina

Если проект серьёзный, нужна максимальная надёжность.

**Selectel** — enterprise-класс. PCI DSS сертификация. Высокий SLA. Техподдержка отвечает быстро. WAF + расширенная DDoS-защита (платно). Цена от 1000 рублей/мес.

Финансовые сервисы, e-commerce, медицинские данные — везде, где нужны PCI DSS и высокая надёжность. Полное соответствие 152-ФЗ.

**VDSina** — мощные серверы. AMD EPYC, высокая частота, до 128 vCPU. Большой лимит трафика/мес включён. 10 Гбит/с локальная сеть между серверами. Дата-центры в Москве и Амстердаме.

API с высоким трафиком, ML inference, масштабирование до 512 ГБ RAM. Цена от 400 рублей/мес.

## Альтернатива PaaS: ONREZA

Русская PaaS платформа. Бесплатно для начала, без кредитки. Поддержка Next.js, Vite, Astro, Nuxt, SvelteKit, React, Vue, Angular. Preview Deployments есть. 152-ФЗ соблюдается.

Оплата рублями.

Миграция занимает 1-2 часа для большинства проектов. Статический сайт — 30-40 минут. Next.js с Docker — час-полтора.
